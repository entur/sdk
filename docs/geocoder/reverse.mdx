# getFeaturesReverse

```typescript
(coords: Coordinates, params?: GetFeaturesReverseParam) => Promise<FeatureCollection<GeocoderTypes.Location>>
```

`getFeaturesReverse` does a reverse lookup of stop places, stations or addresses around the given coordinates.

## Parameters

### coords (`Coordinates`)

The coordinates to do a lookup for. Examples: `{ latitude: 59.909774, longitude: 10.763712 }`.

### params (`ReverseGeocoderParams`) [Optional]

An optional object of parameters to pass to the query.

| Key        | Type                           | Default               | Description                                                                                                                                                                              |
| :--------- | :----------------------------- | :-------------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `point`    | `{ lat: number, lon: number }` |                       | The coordinates to do a lookup for. Examples: `{ lat: 59.909774, lon: 10.763712 }`.                                                                                                      |
| `radius`   | `number`                       | 1                     | Radius (in meters) from the given point to limit results.                                                                                                                                |
| `size`     | `number`                       | No limitation         | Limit the maximum number of results to this number. Valid values are from 1 to 100 inclusive.                                                                                            |
| `layers`   |  `string[]`                    | `["venue","address"]` | The types of places to search for in a comma-separated string. `venue` means stop places and stations, `address` means postal addresses that might not be connected to public transport. |
| `boundary` | `Boundary`                     |                       | Allows filtering by geographical region. See details below.                                                                                                                              |

#### Boundary (object)

| Key           | Type       | Description                                                   |
| :------------ | ---------- | :------------------------------------------------------------ |
| `country`     | `string`   |                                                               |
| `countyIds`   | `string[]` | Use this for filtering on counties.                           |
| `localityIds` | `string[]` | Use this for filtering on municipalities.                     |
| `rect`        | `Rect`     | A bounding box to get features within.                        |
| `circle`      | `Circle`   | A bounding circle surrounding `point` to get features within. |

#### Rect (object)

| Key      | Type     |
| :------- | :------- |
| `minLat` | `number` |
| `minLon` | `number` |
| `maxLat` | `number` |
| `maxLon` | `number` |

#### Circle (object)

| Key      | Type     |
| :------- | :------- |
| `radius` | `number` |

### options (`RequestOptions`) [Optional]

An object containing a subset of `RequestInit` options that's applied to the request.

| Key       | Type          | Description                                                                                                                                       |
| :-------- | :------------ | :------------------------------------------------------------------------------------------------------------------------------------------------ |
| `signal`  | `AbortSignal` | Allows you to communicate with a fetch request and abort it if desired. [Read more](https://developer.mozilla.org/en-US/docs/Web/API/AbortSignal) |
| `headers` | `HeadersInit` | Add custom HTTP headers to the fetch request if desired.                                                                                          |

## Example

```typescript
import { Feature, Point } from 'geojson'
import createEnturService, { GeocoderTypes } from '@entur/sdk'
// or: const createEnturService = require('@entur/sdk').default

const service = createEnturService({
    clientName: 'awesomecompany-awesomeapp',
})

/**
 * Finding locations within 2 km of Oslo City Hall (Rådhuset).
 */
async function example() {
    try {
        const featureCollection = await service.geocoder.reverse({
            point: {
                lat: 60.1283,
                lon: 10.7654,
            },
            boundary: {
                circle: {
                    radius: 2,
                },
            },
        })

        const features: Array<Feature<Point, GeocoderTypes.Location>> =
            featureCollection.features

        console.log(features)
    } catch (error) {
        console.error(error)
    }
}

example()
```
