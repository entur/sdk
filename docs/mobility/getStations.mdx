# getStations

```typescript
(params: getStationsParams, options?: RequestOptions) => Promise<MobilityTypes.Station[]>
```

`getStations` finds micromobility stations within an area surrounding set of coordinates.

This method uses the `stations` query of the [Mobility v2 API](https://developer.entur.org/pages-mobility-docs-mobility-v2).

## Parameters

### params (`getStationsParams`)

| Key          | Type        | Required? | Default | Description                                                                    |
| :----------- | :---------- | :-------- | :------ | ------------------------------------------------------------------------------ |
| `lat`        |  `number`   | Yes       | N/A     | The latitude coordinate to find stations around.                               |
| `lon`        |  `number`   | Yes       | N/A     | The longitude coordinate to find stations around.                              |
| `range`      |  `number`   | Yes       | N/A     | The radius of the search area, surrounding the latitude/longitude coordinates. |
| `count`      |  `number`   | No        |         | The max number of stations to return.                                          |
| `codespaces` |  `string[]` | No        |         | Return only stations of the given code spaces.                                 |
| `operators`  |  `string[]` | No        |         | Return only stations of the given operator IDs.                                |
| `systems`    |  `string[]` | No        |         | Return only stations of the given system IDs.                                  |

### options (`RequestOptions`) [Optional]

An object containing a subset of `RequestInit` options that's applied to the request.

| Key       | Type          | Description                                                                                                                                       |
| :-------- | :------------ | :------------------------------------------------------------------------------------------------------------------------------------------------ |
| `signal`  | `AbortSignal` | Allows you to communicate with a fetch request and abort it if desired. [Read more](https://developer.mozilla.org/en-US/docs/Web/API/AbortSignal) |
| `headers` | `HeadersInit` | Add custom HTTP headers to the fetch request if desired.                                                                                          |

## Example

```javascript
import createEnturClient from '@entur/sdk'
// or: const createEnturClient = require('@entur/sdk').default

const enturClient = createEnturClient({
    clientName: 'awesomecompany-awesomeapp',
})

async function example() {
    try {
        const stations = await enturClient.mobility.getStations({
            lat: 59.95,
            lon: 10.75,
            range: 200,
        })
        console.log(stations)
    } catch (error) {
        console.error(error)
    }
}
```
