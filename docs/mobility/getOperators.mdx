# getOperators

```typescript
(options?: RequestOptions) => Promise<MobilityTypes.Operator[]>
```

`getOperators` finds micromobility operators for city bicycles, e-scooters, shared bikes and cars.

This method uses the `operators` query of the [Mobility v2 API](https://developer.entur.org/pages-mobility-docs-mobility-v2).

### options (`RequestOptions`) [Optional]

An object containing a subset of `RequestInit` options that's applied to the request.

| Key      | Type          | Description                                                                                                                                       |
| :------- | :------------ | :------------------------------------------------------------------------------------------------------------------------------------------------ |
| `signal` | `AbortSignal` | Allows you to communicate with a fetch request and abort it if desired. [Read more](https://developer.mozilla.org/en-US/docs/Web/API/AbortSignal) |

## Example

```javascript
import createEnturService from '@entur/sdk'
// or: const createEnturService = require('@entur/sdk').default

const service = createEnturService({
    clientName: 'awesomecompany-awesomeapp',
})

async function example() {
    try {
        const operators = await service.mobility.getOperators()
        console.log(operators)
    } catch (error) {
        console.error(error)
    }
}
```
