# getStationsById

```typescript
(params: getStationsByIdParams, options?: RequestOptions) => Promise<MobilityTypes.Station[]>
```

`getStationsById` finds micromobility stations with the given station IDs.
If an ID is invalid or not found, it will not be included in the result. Therefore, the resulting
array might be shorter than the argument array.

This method uses the `stationsById` query of the [Mobility v2 API](https://developer.entur.org/pages-mobility-docs-mobility-v2).

## Parameters

### params (`getStationsByIdParams`)

| Key          | Type        | Required? | Default | Description                                |
| :----------- | :---------- | :-------- | :------ | ------------------------------------------ |
| `stationIds` |  `string[]` | Yes       | N/A     | The list of IDs to fetch station data for. |

### options (`RequestOptions`) [Optional]

An object containing a subset of `RequestInit` options that's applied to the request.

| Key       | Type          | Description                                                                                                                                       |
| :-------- | :------------ | :------------------------------------------------------------------------------------------------------------------------------------------------ |
| `signal`  | `AbortSignal` | Allows you to communicate with a fetch request and abort it if desired. [Read more](https://developer.mozilla.org/en-US/docs/Web/API/AbortSignal) |
| `headers` | `HeadersInit` | Add custom HTTP headers to the fetch request if desired.                                                                                          |

## Example

```javascript
import createEnturClient from '@entur/sdk'
// or: const createEnturClient = require('@entur/sdk').default

const enturClient = createEnturClient({
    clientName: 'awesomecompany-awesomeapp',
})

async function example() {
    try {
        const stations = await enturClient.mobility.getStationsById({
            stationIds: [
                'YBE:VehicleSharingParkingArea:731',
                'YOS:VehicleSharingParkingArea:483',
                'some invalid id',
            ],
        })

        // Probably 2 or less, since 'some invalid url' is invalid
        console.log(stations.length)

        return stations
    } catch (error) {
        console.error(error)
    }
}
```
