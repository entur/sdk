# getDeparturesFromStopPlaces

```typescript
(stopPlaceIds: Array<string>, params?: GetDeparturesParams) => Promise<Array<{ id: string, departures: Departure[] } | undefined>>
```

`getDeparturesFromStopPlaces` finds departures from one or more given stop places. Also see `getDeparturesFromStopPlace` for a simpler interface for fetching departures for a single stop.

The method will return an array of objects containing fields for the stop place's `id` and `departures`.

The returned array _can_ contain undefined values. If that's the case, the ID you provided was not valid.

## Parameters

### stopPlaceIds (`Array<string>`)
The IDs of the stop places you are interested in.

### params (`Object`) [Optional]

See the `params` parameter for [`getDeparturesForStopPlace`](/departures/getDeparturesFromStopPlace#params-object-optional).

### options (`RequestOptions`) [Optional]
An object containing a subset of `RequestInit` options that's applied to the request.

| Key      | Type          | Description |
|:---------|:--------------|:------------|
| `signal` | `AbortSignal` | Allows you to communicate with a fetch request and abort it if desired. [Read more](https://developer.mozilla.org/en-US/docs/Web/API/AbortSignal) |
